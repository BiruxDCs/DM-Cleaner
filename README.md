---

## 📌 Description

**2026 DM Cleaner** is an automation tool created by **BiruxDCs** to optimize the management of accumulated chat conversations on messaging platforms.

Its main purpose is to reduce repetitive manual work by allowing users to quickly leave multiple conversations.

---

## ⚙️ Features

- Automated leaving of DM groups
- Fast processing of multiple conversations
- Simple and efficiency-focused design
- Fully auditable source code
- No user data storage
